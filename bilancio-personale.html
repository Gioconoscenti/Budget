Setup Google Sheets Backend

STEP 1 — Crea un nuovo Google Sheet su sheets.google.com. I fogli vengono creati automaticamente.

STEP 2 — Apri Estensioni → Apps Script, incolla il contenuto di Code.gs e salva.

STEP 3 — Esegui setupSheets() una volta dalla console GAS.

STEP 4 — Deploy → Nuova implementazione → Web app
· Esegui come: Me · Accesso: Chiunque
· Copia l'URL (termina con /exec)

STEP 5 — Incolla l'URL nel campo Google Sheets URL e clicca Sync GS.

✓ Dati persistenti su tutti i dispositivi · ✓ Modificabili direttamente nel foglio

Setup Cloudflare Worker + Enable Banking

STEP 1 — Vai su enablebanking.com → crea account → Applications → Create → Personal.
Scarica la chiave RSA privata e copia il tuo Application ID.

STEP 2 — Deploy Cloudflare Worker (usa i file worker.js + wrangler.toml inclusi nel pacchetto):

npm install -g wrangler && wrangler login
wrangler kv:namespace create "SESSIONS"
# → incolla l'ID in wrangler.toml

wrangler secret put PRIVATE_KEY   # incolla chiave .pem
wrangler secret put APP_ID        # incolla Application ID
wrangler deploy
# → https://speselab-proxy.TUO.workers.dev

STEP 3 — In Enable Banking dashboard → Redirect URL:
https://speselab-proxy.TUO.workers.dev/callback

STEP 4 — Incolla l'URL Worker nel campo in alto e clicca Connetti su ogni banca.

📦 I file worker.js, wrangler.toml e README.md con le istruzioni complete sono inclusi nel pacchetto scaricato.
